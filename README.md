# Global Superstore Data Cleaning & Reporting Automation

## Overview
This project automates the data cleaning and reporting process using Python and presents business insights through an interactive Power BI dashboard. The Global Superstore dataset was used to demonstrate data preprocessing, reporting automation, and visualization.

## Objectives
- Clean and preprocess raw data.
- Handle missing values and duplicate records.
- Convert data into appropriate formats.
- Generate automated summary reports.
- Build an interactive Power BI dashboard.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Power BI
- VS Code

## Data Cleaning Steps
- Imported the dataset.
- Checked dataset information.
- Identified missing values.
- Removed duplicate records.
- Converted date columns to datetime format.
- Trimmed unnecessary spaces from text columns.
- Filled missing postal codes.
- Exported the cleaned dataset.

## Automated Reporting
The Python script automatically generates:
- Total rows and columns
- Missing value summary
- Duplicate record count
- Summary statistics
- Total Sales
- Total Profit
- Average Sales

## Power BI Dashboard
The dashboard includes:
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Total Quantity KPI
- Sales by Country (Map)
- Sales by Category
- Profit by Region
- Sales Trend
- Interactive slicers for Market, Category, Region, and Segment
